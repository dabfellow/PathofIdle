# Path Of Idle

Welcome to Path Of Idle, a simple idle game where you progress through different zones, fighting various enemies and leveling up your character.

## Description

Path Of Idle is an idle game where you control a character that fights enemies in different zones. As you defeat enemies, you gain experience and level up, allowing you to allocate points to different attributes such as strength, dexterity, and intelligence.

## Features

- Dynamic enemy types with SVG representations
- Cooldown bars with text indicating remaining cooldown time
- Character leveling and attribute allocation
- Combat log to track actions and events

## Setup

To set up the game on your local machine, follow these steps:

### Option 1: Download from GitHub

1. **Clone the repository:**

    ```sh
    git clone "https://github.com/dabfellow/PathofIdle.git"
    ```

2. **Navigate to the project directory:**

    ```sh
    cd path-of-idle
    ```

3. **Open the [PoI.html](http://_vscodecontentref_/0) file in your web browser:**

    You can simply double-click the [PoI.html](http://_vscodecontentref_/1) file or open it using your browser's "Open File" option.

## How to Play

1. **Character Stats:**

    - Your character starts at level 1 with 50 health and 0 experience.
    - You have 1 available point to allocate to attributes (strength, dexterity, intelligence).

2. **Allocate Points:**

    - Click the "+" button next to an attribute to allocate a point to it.
    - Allocating points to strength increases your health.

3. **Combat:**

    - The combat area displays the current enemy and your character.
    - Click the "Attack" button to attack the enemy.
    - The cooldown bar below the enemy and player indicates the time remaining before the next attack.

4. **Progress Through Zones:**

    - As you defeat enemies, you gain experience and level up.
    - The enemy type changes as you progress through different zones.

5. **Activity Log:**

    - The activity log tracks actions and events during the game.
    - You can filter the log by different types of events (attack, damage, XP gain, etc.).
